수치 조작 메서드
	.width()
	.height()
	
	.innerWidth()	- 내용 너비 + 내부 여백
	.innerHeight() 	- 내용 높이 + 내부 여백
	
	.outerWidth()	- 내용 너비 + 내부 여백 + 경계선
	.outerHeight()	- 내용 높이 + 내부 여백 + 경계선
	
	.offset() -> 선택한 요소가 문서에서 수평/수직으로 얼마나 떨어져있는지 체크
					수평 - left
					수직 - top
			
	.scrollTop() -> 수직 스크롤 양
	.scrollLeft() -> 수평 스크롤 양
